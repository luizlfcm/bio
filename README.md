## 🚀 Deploy no Vercel (Passo a Passo)

### 1. Criar repositório no GitHub
1. Acesse [github.com](https://github.com) e faça login
2. Clique em **New repository**
3. Nomeie como `isabella-bianchi` (ou similar)
4. Deixe **Public** e clique em **Create repository**

### 2. Fazer upload dos arquivos
1. No repositório criado, clique em **Add file → Upload files**
2. Arraste todos os arquivos:
   - `index.html`
   - `foto-profissional_jpeg.jpeg`
   - `foto-profissional-2_jpeg.jpeg`
   - `foto-profissional-3_png.png`
3. Clique em **Commit changes**

### 3. Publicar no Vercel
1. Acesse [vercel.com](https://vercel.com) e faça login com GitHub
2. Clique em **Add New → Project**
3. Selecione o repositório `isabella-bianchi`
4. Clique em **Deploy** (sem alterar nenhuma configuração)
5. Pronto! O site estará no ar em segundos.

## 🌐 Domínio Personalizado (Opcional)

No painel do Vercel:
1. Vá em **Settings → Domains**
2. Adicione seu domínio (ex: `isabellabianchi.com.br`)
3. Siga as instruções de DNS fornecidas
